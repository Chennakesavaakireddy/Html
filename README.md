# Akireddy Tirumala Chennakesavarao | Portfolio Website

## Overview
This is a personal portfolio website for **Akireddy Tirumala Chennakesavarao**, showcasing skills, projects, education, certificates, and contact information. The website demonstrates proficiency in **Python development**, **Web development (HTML, CSS)**, and responsive design principles.

The site is clean, responsive, and built with industry-standard **semantic HTML5** and **modern CSS** practices.

---

## Features
- **About Section:** Personal introduction and profile image.  
- **Skills Section:** Highlighted technical skills in a responsive grid.  
- **Projects Section:** Detailed project descriptions with technologies used and GitHub links.  
- **Education Section:** Complete academic background with percentages/CGPA.  
- **Certificates Section:** Showcase of completed certifications.  
- **Contact Section:** Professional contact information with clickable links for email, GitHub, and LinkedIn.  
- **Responsive Design:** Optimized for desktop, tablet, and mobile devices.  
- **Smooth Scroll:** Navigation links scroll smoothly to each section.  
- **Sticky Header:** The header stays visible while scrolling.  

---

## Technologies Used
- **HTML5** – Semantic and accessible structure  
- **CSS3** – Modern styling with gradients, flexbox, grid, and hover effects  
- **Responsive Design** – Mobile-first and flexible layout  
- **GitHub** – Project hosting and source code management
